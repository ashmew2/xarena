Write a program to accept a string and print it back.
There will be a number of inputs, Process each one and output the result followed by a new line in all cases.

The first line contains N , the number of strings in the input.
The next N lines contain those N strings.


Example :

Input:
	3
	h1
	h2
	h3

Output:
	h1
	h2
	h3
